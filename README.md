  ## First Launch Notes                                                                                                                               
                                                                                                      
  Wordspace is **not yet code-signed**. Your OS will warn you the first time you open it:                                                             
                                                            
  ### macOS — "unidentified developer"                                                                                                                
                                                            
  1. Download and open the `.dmg`, drag Wordspace to Applications                                                                                     
  2. **Right-click** Wordspace in Applications → **Open** → confirm **Open** in the dialog
  3. Future launches work normally (double-click)                                                                                                     
                                                                                                                                                      
  ### Windows — "Windows protected your PC"                                                                                                           
                                                                                                                                                      
  1. Run the `.exe`                                                                                                                                   
  2. Click **More info** → **Run anyway**                                                             
  3. Follow the installer
